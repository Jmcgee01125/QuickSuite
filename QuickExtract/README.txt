
:: -------------------------------------

QuickExtract Version 1.0

:: -------------------------------------

QuickExtract allows quickly extracting the audio from a video file.

:: -------------------------------------

To use, drag the video file onto QuickExtract.bat.

The output will be saved as <source file>_qe.<filetype>

:: -------------------------------------

By opening QuickExtract.bat, one variable becomes available:
	Filetype - Set the audio format to be used in the output (default: mp3).
