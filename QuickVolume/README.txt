
:: -------------------------------------

QuickVolume Version 1.0d

:: -------------------------------------

QuickVolume allows changing the audio volume of a video file without re-encoding the whole thing.

:: -------------------------------------

To use, drag the video file onto QuickVolume.bat, then type a percentage to scale the volume by (but leave out the percent sign, e.g. "125" for 125%).

The output will be saved as <source file>_qv.<source extension>

:: -------------------------------------

By opening QuickVolume.bat, one variable becomes available:
	abr - Optional override for the output audio bitrate, useful for webm (default: ).
